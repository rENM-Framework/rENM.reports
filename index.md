This is the archive repository for code, data, and other resources related the development of the rENM framework. Collectively, this set of rENM R packages provide tools to create retrospective ecological niche models (rENMs) and perform climatic suitability analyses for North American bird species.

The rENM.reports R package provides functions for summarizing and reporting the outputs created by rENM.data and rENM.analysis packages.

The rENM framework offers a reproducible, modular workflow that supports high-volume rENM pipelines and automated species-level analyses to support basic research and conservation practices. The codebase implements the rENM workflows described in the following papers:
